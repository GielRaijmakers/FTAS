This excelsheet is a first setup for making testautomation more easy for FTAS. 
FTAS is is build so it can handle easy scripts. THis script can be made by this excel sheet by exporting the test.

The test is build out of keywords and arguments. 
logon(name;password)
Logon is a keyword, the other are arguments. 

In the first column (A) all cells are filled with a dropdown with a list of keywords. 
When selecting a keywor, the arguments are placed in the row above the keyword.

With a rightclick the cell below a argument can be filled with a predefined value like <click>.

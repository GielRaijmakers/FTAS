Welcome to FTAS v0.1 
This framework is made for functional testautomation with Ruby, SikuliX.
 
As a bonus, there is also a Excel sheet to make things more easy for testers. 

The Sikuli framework exists out of the following elements: 
- Sikuli
-- Framework(contains global functions for handeling objects)
-- Repo (this contains all images from sikuli)
-- Handeling (this handels all actionwords from the framework).

- Excel
--framework (VBA code with global functions for handeling actions)
--project (VBA code for handeling project specific actions)
--Access database that contains all actionwords and objects belonging to the different projects. 

Please read the wiki on github for extended information on all code and functionality.
